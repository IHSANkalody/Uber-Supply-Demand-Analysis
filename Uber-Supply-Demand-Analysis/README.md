### &nbsp;                **Uber Ride Supply–Demand Analysis**



##### Project Overview

This project analyzes Uber ride request data to identify supply-demand gaps, peak-hour shortages, and cancellation patterns.



##### Objective

&nbsp;• Identify high-demand time slots

&nbsp;• Analyze driver unavailability

&nbsp;• Understand cancellation trends

&nbsp;• Provide business recommendations

##### 

##### Tools \& Technologies

&nbsp;• Python

&nbsp;• Pandas

&nbsp;• NumPy

&nbsp;• Matplotlib  

&nbsp;• Seaborn

&nbsp;



##### Dataset

Uber ride request dataset containing:

&nbsp;• Request ID

&nbsp;• Pickup Point

&nbsp;• Status

&nbsp;• Request Time

&nbsp;• Drop Time



#### Key Insights

&nbsp;• Highest demand observed during evening hours

&nbsp;• Maximum cancellations occurred at early morning

&nbsp;• Airport pickup had higher driver unavailability



#### Business Impact

• Improves ride fulfillment rate and reduces supply-demand gap.
